tag 资源志愿征集

因开发者精力有限，无法全面收集并调试项目全量 tag，现特向所有项目使用者志愿征集调试完成的 tag。

征集到的 tag 会统一整理，后续将以两种形式发放：

  1. 整合至项目新版本打包发布
  2. 单独打包为 imageData.zip，发布至 GitHub 或交流群内

有需求的朋友可直接加入交流群，免费下载使用。

imageData 模块说明

本项目基础功能支持自定义图片配置，同名的图片文件与 txt 文件为一个独立配置单元，在该单元下可对 tags 进行任意增删改查操作。

投稿类型包括但不限于如下：

  • 01 - 人物
  • 02 - 画风
  • 03 - 动作

  • 05 - 服饰
